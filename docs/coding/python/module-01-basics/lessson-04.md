## Lesson 2: Variables & Data Types

### Concept
Variables are like labeled boxes that store information.

### Live Demo
&lt;py-script&gt;
# Creating variables
student_name = "Alex"
student_age = 12
is_coding_fun = True

print("Student Profile:")
print(f"Name: {student_name}")
print(f"Age: {student_age}")
print(f"Loves coding: {is_coding_fun}")
&lt;/py-script&gt;
