    then move to Python for deeper programming skills.

## How Tracks Are Structured

Every module in every track follows the same pattern:

```
Module
├── index.md          ← Module overview and lesson list
├── lesson-01.md      ← Concept + code + exercise + quiz
├── lesson-02.md
├── lesson-03.md
└── projects/
    ├── index.md      ← Project hub
    ├── mini-01.md    ← After lesson 01
    ├── mini-02.md    ← After lesson 02
    ├── mini-03.md    ← After lesson 03
    └── capstone.md   ← After all lessons
```

!!! success "The rule"
    Complete a lesson → build the mini project → move to the next lesson.
    Complete all lessons → build the capstone.
